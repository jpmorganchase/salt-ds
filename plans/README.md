## Findings considered and rejected

- **`bundle:css` background jobs can escape the build**: rejected. Yarn 4's portable shell treats `&` as an awaited parallel job and reports after all jobs finish; the script is terse but the claimed missing wait is not supported.
- **Site search `dangerouslySetInnerHTML` is an external-input vulnerability**: rejected as a security finding because the configured index sources are maintainer-controlled local docs and repository releases. Defense-in-depth escaping/sanitization remains reasonable if the trust boundary changes.
- **All Yarn peer warnings are actionable defects**: rejected. Only direct contradictions with shipped behavior or the tested graph were planned; transitive/optional peer noise was not promoted without reachability evidence.
- **AriaAnnouncer still leaks timers**: rejected as stale. The current provider tracks and clears handles, and the focused regression exists.
- **The site needs a duplicate GitHub Actions build**: rejected on maintainer evidence. Vercel runs the production site build on every pull request; only document or revisit that gate if it stops blocking merges or diverges from the production command.
- **ElectronWindow lifecycle/security repair**: rejected because the maintainer identified `ElectronWindow` as the obsolete mechanism. Prefer removal/deprecation work if the old path is revisited.
- **ColorChooser Swatch accessibility repair**: rejected from this plan set because ColorChooser remains an unprioritized lab component. Reconsider before promotion or a stable accessibility commitment.
- **StaticList semantic alignment**: rejected as duplicate ownership; the maintainer is handling it in a separate workstream.
- **Vitest security upgrade**: rejected as duplicate ownership; another workstream is already handling the direct dependency and lockfile change.
- **Formal React 19/support-policy project**: rejected for now. A daily workflow installs React canary, builds packages, and runs Cypress; reopen only for a concrete stable-major failure or support-floor decision.
- **JPM Brand theme graduation**: rejected for the current planning horizon at maintainer direction.
- **All remaining lab runtime fixes**: delayed at maintainer direction. This includes Portal SSR/default-window behavior, lab ResizeObserver consolidation, and responsive OverflowReducer optimization; active plans may not pull them back into scope indirectly.
