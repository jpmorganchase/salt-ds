Added `SegmentedButtonGroup` to core.
`SegmentedButtonGroup` shows a list of actionable buttons, flush with separators between them.

```tsx
return (
  <SegmentedButtonGroup>
    <Button variant={variant}>Button</Button>
    <Button variant={variant}>Button</Button>
    <Button variant={variant}>Button</Button>
  </SegmentedButtonGroup>
);
```
