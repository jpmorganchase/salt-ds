Added Heatmap chart support. The `saltColorAxis` option on `useChart` can now be used with heatmaps so color scales align with Salt design tokens (continuous gradients, data classes, or discrete ranges).

To see an example configuration, see the Heatmap Options example in the Chart example section, and the Color axis section of the Chart usage docs.
