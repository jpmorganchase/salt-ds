<KeyboardControls>
<KeyboardControl keyOrCombos="Tab">

If focus is above the List, Tab key moves focus into:

- The last selected item;
- The last item with focus or;
- The first item in the List

</KeyboardControl>
<KeyboardControl keyOrCombos="Shift + Tab">

If focus is below the List, Tab key moves focus into the list item
which last had focus; or the first item in the List

</KeyboardControl>
<KeyboardControl keyOrCombos="Enter / Space">

Single-select List: when focus is on a listed item, Enter or Space
keys press, select the item and deselects the previously selected
item. If `deselectable` is active and focus is on a
selected item, it's deselected.

</KeyboardControl>
<KeyboardControl keyOrCombos="Arrow up">

Moves focus to the previous item in the List. If focus is on the first
item, the next Up Arrow key press should NOT wrap focus to the last item

</KeyboardControl>
<KeyboardControl keyOrCombos="Arrow down">

Moves focus to next item in the List. If focus is on the last item, the
next Down Arrow key press should NOT wrap focus back to the first item

</KeyboardControl>
<KeyboardControl keyOrCombos="Page up">

Moves the highlight a number of positions up the List, based on the
number specified in `displayedItemCount`.

</KeyboardControl>
<KeyboardControl keyOrCombos="Page down">

Moves the highlight a number of positions down the list, based on the
number specified in `displayedItemCount`.

</KeyboardControl>
<KeyboardControl keyOrCombos="Home">

Moves focus to the first list item.

</KeyboardControl>
<KeyboardControl keyOrCombos="End">

Moves focus to the last list item.

</KeyboardControl>
</KeyboardControls>
