## Single-select

The default list allows a user to make a single selection from the options available.

</LivePreview>

<LivePreview componentName="list-box" exampleName="Multiselect">

## Multiselect

When `multiselect={true}`, users can select multiple items from the list box options.

</LivePreview>

<LivePreview componentName="list-box" exampleName="GroupedOptions">

## Grouped options

Wrap options in a `OptionGroup` component to group them together. Use the `label` prop to provide a label for the group.

</LivePreview>

<LivePreview componentName="list-box" exampleName="DisabledOption">

## Disabled option

Use the `disabled` prop on the `Option` component to disable an option. Disabled options are not selectable and are visually distinct from selectable options.

</LivePreview>

<LivePreview componentName="list-box" exampleName="Borderless">

## Borderless

By default, list box has a border. Use the `borderless` prop to remove the border. This is useful when the list box is used in a container with a border. See the [list builder pattern](/salt/patterns/list-builder) for an example of this.

</LivePreview>

<LivePreview componentName="list-box" exampleName="Scrolling">

## Scrolling

The list box will scroll when the number of options exceeds the height of the list box.

</LivePreview>
