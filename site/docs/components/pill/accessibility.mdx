## Keyboard interactions

<KeyboardControls>
  <KeyboardControl keyOrCombos="Enter / Space">
    When focus is on the pill, this action activates it.
  </KeyboardControl>
  <KeyboardControl keyOrCombos="Tab">

    - Tab sets focus on the entire pill.
    - When the pill has focus, Tab moves focus out of the pill to next component in tab order.

  </KeyboardControl>
  <KeyboardControl keyOrCombos="Shift + Tab">
    When the pill has focus, this action moves focus out of the pill to the previous component in the tab
    order.
  </KeyboardControl>
</KeyboardControls>

### Selectable pill group

<KeyboardControls>
  <KeyboardControl keyOrCombos="Space">
    Space changes the state of the checkbox, select or deselect.
  </KeyboardControl>
  <KeyboardControl keyOrCombos="Tab">
    If focus is above the pill group, Tab moves focus to one of the following
    options depending on previous user interaction:

    - The last selected option
    - The first option in the list

  </KeyboardControl>
  <KeyboardControl keyOrCombos="Shift + Tab">
    If focus is after the pill group, this action moves focus to one of the following options depending on previous user interaction:
    
    - The last selected option
    - The last option in the list
  </KeyboardControl>
  <KeyboardControl keyOrCombos="Left arrow / Up arrow">
    Left arrow or Up arrow moves focus to the previous pill in the group. Focus doesn't wrap
  </KeyboardControl>
  <KeyboardControl keyOrCombos="Right arrow / Down arrow">
    Right arrow or Down arrow moves focus to the next pill in the group. Focus doesn't wrap.
  </KeyboardControl>
  <KeyboardControl keyOrCombos="Home">
    When focus is within the group, Home moves focus to the first option.
  </KeyboardControl>
  <KeyboardControl keyOrCombos="End">
    When focus is within the group, End moves focus to the last option.
  </KeyboardControl>
</KeyboardControls>
