## Keyboard interactions

<KeyboardControls>
  <KeyboardControl keyOrCombos="Enter / Space">
    When focus is on the pill, this action activates it.
  </KeyboardControl>
  <KeyboardControl keyOrCombos="Tab">

    - Tab sets focus on the entire pill.
    - When the pill has focus, Tab moves focus out of the pill to next component in tab order.

  </KeyboardControl>
  <KeyboardControl keyOrCombos="Shift + Tab">
    When the pill has focus, this action moves focus out of the pill to the previous component in the tab
    order.
  </KeyboardControl>
</KeyboardControls>

### Selectable pill group

<KeyboardControls>
  <KeyboardControl keyOrCombos="Space">
    If the pill is in a selectable group, Space changes the state of the pill, select or deselect.
  </KeyboardControl>
  <KeyboardControl keyOrCombos="Tab">
    If focus is above the pill group, Tab moves focus to one of the first pill in the group.
  </KeyboardControl>
  <KeyboardControl keyOrCombos="Shift + Tab">
    If focus is above the pill group, Tab moves focus to one of the last pill in the group.
  </KeyboardControl>
</KeyboardControls>
