## Import

To import `Carousel`, `CarouselSlide`, `CarouselSlider`, `CarouselHeader` and `CarouselControls` from the lab Salt package, use:

```js
import {
  Carousel,
  CarouselSlide,
  CarouselSlider,
  CarouselHeader,
  CarouselControls,
} from "@salt-ds/lab";
```
