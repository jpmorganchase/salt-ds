<LivePreviewControls>
<LivePreview componentName="banner" exampleName="Static" >

Static banners display information to the user and are non-functional. These types of Banner have no interactive elements.

</LivePreview>

<LivePreview componentName="banner" exampleName="Interactive" >

The interactive Banner includes at least one function allowing users to dismiss or resolve it. However, it can include additional functions as needed (i.e., links, buttons, etc).

Use BannerActions to add actions like close, refresh, etc.

</LivePreview>

<LivePreview componentName="banner" exampleName="Info" >

You should use the information Banner to draw attention to information your user was unaware of, for example, a link to updated terms and conditions.

</LivePreview>

<LivePreview componentName="banner" exampleName="Error" >

If your users have a problem that prevents them from completing a task, such as a system error or technical failure, use the error Banner to tell them how to resolve it.

</LivePreview>

<LivePreview componentName="banner" exampleName="Warning" >

Use the warning Banner to let users know about an issue that won't prevent them from continuing the task, but may cause errors if it's not addressed, such as a restriction due to entitlements or current system status.

</LivePreview>

<LivePreview componentName="banner" exampleName="Success" >

Use the success Banner to reassure your users that an event or process related to the user's current task has been successfully completed.

</LivePreview>

</LivePreviewControls>
