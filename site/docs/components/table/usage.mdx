### Best practices

- Only use the `Table` component for when you need to present simple data.

## Using the component

### When to use

- When you need to present data in a structured, tableau format.

### When not to use

- When you need to show complex data, instead use [AG Grid with Salt](../ag-grid-theme/examples)
