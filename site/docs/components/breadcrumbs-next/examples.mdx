## Wrapped

Use `wrap` when the breadcrumb trail should remain visible and the page layout can support additional height.

<LivePreview componentName="breadcrumbs-next" exampleName="Wrapped" />

## Disclosure collapsed

Set `maxItems` to collapse middle breadcrumb levels into a disclosure when horizontal space is limited. The hidden levels are listed in hierarchy order.

<LivePreview
  componentName="breadcrumbs-next"
  exampleName="DisclosureCollapsed"

## Router integration

Use `render` to adapt breadcrumb links to a routing library such as React Router or Next.js. The same renderer is used for visible links and links inside the collapsed disclosure.

<LivePreview componentName="breadcrumbs-next" exampleName="RouterIntegration" />

## Inline expansion

Set `collapseMode="expand"` to let users reveal hidden breadcrumb levels inline when the full path is useful in the page context.
