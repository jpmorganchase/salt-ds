`BreadcrumbsNext` renders a `<nav>` element with `aria-label="Breadcrumb"` and an ordered list of `BreadcrumbNext` items. Ensure each breadcrumb level has meaningful text content and that the current level is clearly distinguishable from preceding levels once interactive behavior is implemented.

For broader accessibility guidance, refer to the [Breadcrumbs pattern](/salt/patterns/breadcrumbs).
