## Default

`Tree` displays hierarchical data as nested, expandable items. Each `TreeNode` can contain child nodes to create a multi-level structure.

<LivePreview componentName="tree" exampleName="Default" />

## With icons

Use the `icon` prop on `TreeNode` to display an icon alongside the label. Icons help users quickly identify the type of content each node represents.

### Best practices

- Use consistent icons across similar node types (e.g., folder icons for parent nodes, document icons for leaf nodes).

<LivePreview componentName="tree" exampleName="WithIcons" />

## Multiselect

Set `multiselect` to enable multiple node selection with checkboxes. This provides clear visual indication of selection state and supports keyboard shortcuts like Shift + Arrow keys and Ctrl/Cmd + A.

<LivePreview componentName="tree" exampleName="Multiselect" />

## Controlled

Use the `expanded`, `onExpandedChange`, `selected`, and `onSelectionChange` props to control the tree's expansion and selection state externally.

<LivePreview componentName="tree" exampleName="Controlled" />

## Disabled

Use the `disabled` prop on `Tree` to disable the entire tree, or on individual `TreeNode` components to disable specific nodes. Disabled nodes cannot be selected, expanded, or interacted with.

<LivePreview componentName="tree" exampleName="Disabled" />
