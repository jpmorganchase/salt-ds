## Default

## Collapsed

The parent-child layout component changes to a collapsed view once it approaches a specified breakpoint. You can customize this with the `collapsedAtBreakpoint` prop.
The view that is visible when in a stacked state can be customized using the `visibleView` prop.

</LivePreview>
