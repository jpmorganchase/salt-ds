## Variants

There are three variants of the Salt AG Grid theme: primary (default), secondary and zebra. To use one of the non-default variants, add an additional class name to the container element above `<AgGridReact/>`.

The class names are:

- `ag-theme-salt-variant-secondary`
- `ag-theme-salt-variant-zebra`

## Cell editors

Salt provides styling for a few ag grid [built in cell editor](https://www.ag-grid.com/react-data-grid/provided-cell-editors/) types to best match Salt theme. This is only enabled when `editable-cell` is applied in `cellClass` together with corresponding `ag*CellEditor` specificed in `cellEditor` of `ColDef`, so this won't affect your own custom cell editors.

Note that [Rich Select Cell Editor](https://www.ag-grid.com/react-data-grid/provided-cell-editors-rich-select/) requires [enterprise license](https://www.ag-grid.com/react-data-grid/licensing/) which offers more functionality than [Select Cell Editor](https://www.ag-grid.com/react-data-grid/provided-cell-editors-select/).

You can also pair chosen editor with provided [column filters types](https://www.ag-grid.com/react-data-grid/filtering/#column-filter-types) `ag*ColumnFilter` in `filter` of `ColDef`.
