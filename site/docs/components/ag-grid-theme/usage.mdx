### Using AG Grid Theme

#### When to use AG Grid Theme

When you want to use the Salt theme with AG Grid.

#### When not to use AG Grid Theme

When you want to use a component that is not related to AG Grid.

### Import

{/* Update the text and code snippet below as needed: */}

To install the `@salt-ds/ag-grid-theme` package, run the command appropriate to your environment:

```js
yarn add @salt-ds/ag-grid-theme
```

```js
npm install @salt-ds/ag-grid-theme
```

#### Import ag-grid styles

You need to import ag-grid styles along with our themes.

```
import "ag-grid-community/styles/ag-grid.css";
```

#### Import Salt AG Grid theme

To import the Salt AG Grid theme, use:

```
import "@salt-ds/ag-grid-theme/salt.css";
```

For light mode, use these CSS class name `.ag-theme-salt-light`.

For dark mode, use these CSS class name `.ag-theme-salt-dark`.

For `compact` version, use these CSS class names:

```
.ag-theme-salt-compact-light
.ag-theme-salt-compact-dark
```

For variants, which are available only on salt theme, use these CSS class names:

```
.ag-theme-salt-variant-secondary
.ag-theme-salt-variant-zebra
```

For editable fields, add the class name `.editable-cell` to the `cellClass` array in your column definition.

For fields displaying numeric values, add the class name `.numeric-cell` to the `cellClass` array in your column definition.

#### Import UITK AG Grid theme

UITK AG Grid theme contains CSS files with a theme that resembles the Toolkit V1 Data Grid component; use this when you would like to keep the Toolkit V1 theme.

To import the UITK AG Grid theme, use:

```js
import "@salt-ds/ag-grid-theme/uitk.css";
```

For light mode, use these CSS class name `.ag-theme-uitk-light`.

For dark mode, use these CSS class name `.ag-theme-uitk-dark`.

For `compact` version, use these CSS class names:

```js
.ag-theme-uitk-compact-light
.ag-theme-uitk-compact-dark
```
