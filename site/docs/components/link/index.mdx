<Callout title="Primitive choice">

Use `Link` for navigation to a destination. If the primary intent is to trigger
work in the current view, use [`Button`](/salt/components/button) instead. For
the broader decision model, see
[Choosing the right primitive](/salt/getting-started/choosing-the-right-primitive).

</Callout>
