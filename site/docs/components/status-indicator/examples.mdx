  <LivePreview componentName="status-indicator" exampleName="Status">
    Use the `status` property to change the status displayed. This will affect
    both the Icon and color of the `StatusIndicator`. The status indicator
    offers four severity levels that set a distinctive indicator and color.
  </LivePreview>
  <LivePreview componentName="status-indicator" exampleName="Size">
    The size of the StatusIndicator can be controlled with the `size` property.
    This number will be used as a multiplier of the base size, in the same way
    as the `Icon` components.
  </LivePreview>
