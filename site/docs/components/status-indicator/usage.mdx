### When to use Status Indicator

- An error indicator should be used when there is critical information preventing an action.

- A warning or info (depending on the severity) indicator should be used when there is important, but non blocking, information to be conveyed to the user.

- Use a success indicator to confirm that an action has been completed.

### Import

To import Status Indicator from the core Salt package, use:

```
import { StatusIndicator } from "@salt-ds/core";
```
