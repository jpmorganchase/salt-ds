### Using the Avatar component

- Avatars may appear in any number of places.  

- The same image, initials, or placeholder image should appear for a given object instance throughout the experience.  

### Import

To import Avatar from the core Salt package, use:

```
import { Avatar } from "@salt-ds/core";
```
