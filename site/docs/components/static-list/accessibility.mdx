## Keyboard interactions

<KeyboardControls>
<KeyboardControl keyOrCombos="Tab">

- If focus is above the Static List, Tab sets focus on the first interactive element (like a button) if present in the List.

</KeyboardControl>
<KeyboardControl keyOrCombos="Shift + Tab">

- If focus is below the List, Tab sets focus on the last interactive element (like a button) if present in the List.

</KeyboardControl>
</KeyboardControls>
