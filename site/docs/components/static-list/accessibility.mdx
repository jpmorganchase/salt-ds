## Keyboard interactions

<KeyboardControls>
<KeyboardControl keyOrCombos="Tab">

- If focus is above the Static List, Tab sets focus on the first action button (interactive element) if present in the list.

</KeyboardControl>
<KeyboardControl keyOrCombos="Shift + Tab">

- If focus is below the List, this action sets focus on the last action button (interactive element) if present in the list.

</KeyboardControl>
</KeyboardControls>
