## Using the component

- Use a mega menu when top-level navigation needs grouped links and richer content than a simple menu.
- Keep group headers clear and concise, and organize items by user task.
- Use custom regions sparingly to highlight supporting information without overwhelming navigation.

## Import

To import `MegaMenu` and related components from the lab package, use:

```js
import {
  MegaMenu,
  MegaMenuTrigger,
  MegaMenuContainer,
  MegaMenuSection,
  MegaMenuGroup,
  MegaMenuHeader,
  MegaMenuItem,
  MegaMenuCustomRegion,
  MegaMenuLinkRow,
} from "@salt-ds/lab";
```
