### Keyboard interactions

<KeyboardControls>
<KeyboardControl keyOrCombos="Tab">

- If focus is above the combo box, Tab sets focus on the component. When focused, combo box has a visible text cursor and the list is opened.
- If focus is on combo box, Tab moves focus to the next focusable element in the tab order.

</KeyboardControl>
<KeyboardControl keyOrCombos="Shift + Tab">

- If focus is below the combo box, this action sets focus on the component.
- If focus is on combo box, this action moves focus to the previous focusable element in the tab order.

</KeyboardControl>
<KeyboardControl keyOrCombos="Enter">

- If the list is closed and combo box has focus, Enter opens the list. Focus remains on combo box.
- If the list is open and focus is on a list item, Enter makes a selection.

</KeyboardControl>
<KeyboardControl keyOrCombos="Space">

 - Space types a space character into the input field.

</KeyboardControl>
<KeyboardControl keyOrCombos="Escape">

- If the list is open, closes the list without making a selection, existing text in input field remains. Focus remains on combo box.
- If the list is closed, does nothing. Focus remains on combo box.

</KeyboardControl>
<KeyboardControl keyOrCombos="Up arrow">

- Up arrow moves visual focus to the previous item in the list. Focus does not wrap.

</KeyboardControl>
<KeyboardControl keyOrCombos="Down arrow">

- Down arrow moves visual focus to the next item in the list. Focus does not wrap.

</KeyboardControl>
<KeyboardControl keyOrCombos="Home">

- When focus is within the combo box, Home moves focus to the first list item.

</KeyboardControl>
<KeyboardControl keyOrCombos="End">

- When focus is within the combo box, End moves focus to the last list item.

</KeyboardControl>
</KeyboardControls>
