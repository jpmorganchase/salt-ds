### Keyboard interactions

<KeyboardControls>
<KeyboardControl keyOrCombos="Tab">

Tab moves focus to a target element in the correct tabbing order on a page.

</KeyboardControl>
<KeyboardControl keyOrCombos="Shift + Tab">

When focus is on a target element, this moves focus to the previous interactive element within the tab order.

</KeyboardControl>
<KeyboardControl keyOrCombos="Enter">

If the target element has focus, Enter performs an action.

</KeyboardControl>
</KeyboardControls>
