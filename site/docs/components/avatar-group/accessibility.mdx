## Best practices

Ensure each `Avatar` within the group has a meaningful accessible name so screen reader users can identify individual members.

The surplus indicator is announced as “N more” by default so that assistive technology users are aware that additional members are hidden. If you provide a custom `renderSurplus`, make sure the surplus element remains accessible and communicates the number of hidden members.

When the Avatar Group is interactive, provide a descriptive accessible name (for example, “Team members”) so screen reader users understand what action will occur when it is activated.
