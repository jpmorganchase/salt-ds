## Bold

Bold tags are typically used when you want to emphasize or highlight specific content. Tag will have bold emphasis by default.

  </LivePreview>

Display icons inside a tag by passing an icon as children to visually reinforce the theme or grouping of the tag and for quick recognition.

## Subtle

Subtle tags are more suitable for providing additional information or context without drawing too much attention.
you can use subltle tags by passing it as `emphasis` prop.

You can set a border in a subtle tag by passing `bordered` as a prop to the component.
