### Using the component

Place it in an application window, widget, data visualization or panel.

#### When to use

- When you want to focus the user's attention on the content of an overlaying component within a specific viewport or content area.
- Place it in an application window, widget, data visualisation or panel.

#### When not to use

When you don't need to focus the user's attention on the overlaying component.

### Import

To import `Scrim` from the lab Salt package, use:

```js
import { Scrim } from "@salt-ds/lab";
```
