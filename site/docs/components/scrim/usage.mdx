### Using the component

Place it in an application window, widget, data visualization or panel.

#### When to use

To focus the user's attention on the content of an overlaying component within a specific viewport or content area.

### Import

To import `Scrim` from the lab Salt package, use:

```js
import { Scrim } from "@salt-ds/lab";
```
