## Variants

- Group buttons of the same varinat to indicate that they are related.
- Buttons within a single segmented button group should all be of the same varinat.
- See the [button docs](/salt/components/button/accessibility). on which varinat to use.

</LivePreview>

## Icon Only

- Avoid using icons that are very similar in the same segmented button group.
- Use Tooltip on an icon-only button to describe the action as per the [button guidance](/salt/components/button/examples#icon-only).
