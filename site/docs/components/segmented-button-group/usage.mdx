## Using the component

Within a single segmented button group, only use buttons of the same variant.

### When to use

Use the segmented button group to display a list of actionable buttons, flush with separators between them.

## Import

To import `SegmentedButtonGroup` from the lab Salt package, use:

```js
import { SegmentedButtonGroup } from "@salt-ds/lab";
```
