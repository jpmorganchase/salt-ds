## Using the component

- Within a single button group only use buttons of the same variant

### When to use

Use the Segmented Button Group to display a list of actionable buttons, flush with separators between them.

## Import

To import `SegmentedButtonGroup` from the core Salt package, use:

```js
import { SegmentedButtonGroup } from "@salt-ds/lab";
```
