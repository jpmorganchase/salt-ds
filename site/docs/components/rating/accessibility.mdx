  - If focus is inside the rating, this action moves focus out of the component.

  rating, this action move focus into the rating. Focus is set to the selected

<KeyboardControl keyOrCombos="Right arrow / Down key">
  - This action moves focus to the next rating item in the rating component. -
  If focus is on the last rating item, this action wraps focus to the first
  rating item.
</KeyboardControl>

  <KeyboardControl keyOrCombos="Left arrow / Up key">
    - This action moves focus to the previous rating item in the Rating component.
    - If focus is on the first rating item, this action wraps focus to the last rating item.
  </KeyboardControl>
</KeyboardControls>
