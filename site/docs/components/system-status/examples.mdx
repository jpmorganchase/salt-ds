## Placement

System status alerts are displayed at the very top of the screen, extending across the full width of the page. 

They appear at the same level as the main content, causing any subsequent content to shift downward to make room. The alert remains visible and continues to occupy space until it is updated or removed by the system (e.g., when the status changes), at which point the content below returns to its original position.
