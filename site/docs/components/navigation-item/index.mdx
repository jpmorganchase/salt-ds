<Callout variant="secondary" status="warning">

Use [`VerticalNavigation`](/salt/components/vertical-navigation) for new
vertical navigation work. `NavigationItem` can remain in existing vertical
implementations during migration, but we recommend moving to
`VerticalNavigation` because it provides a more structured approach and
`NavigationItem` vertical usage will be deprecated in a future release.

</Callout>
