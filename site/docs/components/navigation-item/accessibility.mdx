### Keyboard interactions

#### Horizontal navigation

<KeyboardControls>
  <KeyboardControl keyOrCombos="Tab">
  
  - Tab moves focus into the first navigation item in the group. Sets focus to the entire component.
  
  
  - When the navigation item has focus, Tab moves focus out of the navigation item to next component in tab order.
  
  </KeyboardControl>
  <KeyboardControl keyOrCombos="Shift + Tab">
  
  - When the navigation item has focus, this action moves focus out of the navigation item to previous component in tab order.
  
  </KeyboardControl>
  <KeyboardControl keyOrCombos="Enter / Space">
  
  - This action selects the focused navigation item, or triggers the navigation item to reveal expanded list or mega-menu.
  
  </KeyboardControl>

#### Vertical navigation

  <KeyboardControl keyOrCombos="Tab">
  
  - Tab moves focus into the first navigation item in the vertical group. Sets focus to the entire component.
  
  - If a navigation item is selected before the navigation item receives focus, Tab sets focus on the selected item.
  
  - When the navigation item has children, second key press moves focus on to the arrow icon.
  
  - When the navigation item has focus, Tab moves focus out of the navigation item to the next component in tab order.
  
  </KeyboardControl>
  <KeyboardControl keyOrCombos="Shift + Tab">
  
  - If focus is below the navigation item, this action moves focus to the currently selected navigation item in the navigation item component.
  
  - If focus is on an item inside the navigation group, this action moves focus out of the navigation to the previous focusable element in the tab order.
  
  </KeyboardControl>
  <KeyboardControl keyOrCombos="Enter / Space">
  
  - This action selects the focused navigation item.
  
  - This action triggers the navigation item to reveal nested levels within the navigation.
  </KeyboardControl>
</KeyboardControls>
