Opacity refers to the degree of transparency or visibility of a design element. It allows for layering, creating depth and hierarchy, and can help provide visual interest to an interface.

## Opacity scale

| Token       | Opacity |
| ----------- | ------- |
| `--salt-opacity-1` | 0.15    |
| `--salt-opacity-2` | 0.25    |
| `--salt-opacity-3` | 0.40    |
| `--salt-opacity-4` | 0.70    |
