# Icon

Icon is an important visual representation of an application, a capability, or some other concept or specific entity with meaning on a interface. It is expected to help user to recognize and understand a content or an action regardless languages or culture boundaries.
We have 3 sizes of icon set for users to choose in terms of different platforms or device size.

If an icon is interactive, use [Button](?path=/docs/documentation-core-button--button-variants) (`CTA`, `Primary` or `Secondary` variant).
