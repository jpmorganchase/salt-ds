# Status icon

A status icon can be used on its own or within another component (e.g. `Tooltip`) to help convey a message.

<Canvas>
  <Story id="core-statusicon--default-status-icon" />
</Canvas>

## How you can use status icon

Use this component by passing in your chosen `status` from the available options.

You can also customize the icon's `size` and pass custom props to the underlying SVG element via `SVGProps`.

## Configuring status icon

### API

## Features

### Status icons

The status icon offers four severity levels that set a distinctive icon and color.

<Canvas>
  <Story id="core-statusicon--all-status-icons" />
</Canvas>

### Use when...

An `error` icon should be used when there is critical information preventing an action.

A `warning` or `info` (depending on the severity) icon should be used when there is important, but non blocking, information to be conveyed to the user.

Use a `success` icon to confirm that an action has been completed.
