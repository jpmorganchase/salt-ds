# State icon

A state icon can be used on its own or within another component (e.g. `Tooltip`) to help convey a message.

<Canvas>
  <Story id="core-stateicon--default-state-icon" />
</Canvas>

## How you can use state icon

Use this component by passing in your chosen `state` from the available options.

You can also customise the icon's `size` and pass custom props to the unterlying svg element via `SVGProps`.

# API

```
import { StateIcon } from "@jpmorganchase/uitk-core";
```
