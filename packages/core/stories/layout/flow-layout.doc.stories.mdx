# Flow Layout

Dictates a vertical or horizontal order and direction for UI elements.
Builds from flex layout

Flow is responsive by default and will break into one column as soon as there is not enough space.

<Canvas>
  <Story id="core-layout-flowlayout--default-flow-layout" />
</Canvas>

Height is driven by child items,
Visual seperators can be turned on/off (default) between items
