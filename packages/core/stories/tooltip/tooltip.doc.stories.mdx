# Tooltip

A Tooltip provides guidance to the user when something requires more explanation or context, or to provide more information about an error or warning condition. A Tooltip will typically be displayed after the userâ€™s mouse hovers over the target element for a certain amount of time.

## Features

### Status

Use the `status` prop to choose between `error`, `warning`, `success` and `info`.

<Canvas>
  <Story id="core-tooltip--status" decorators={[withFlexGap]} />
</Canvas>

### Hide arrow

Use `hideArrow` prop if you'd like to display the tooltip without the arrow.

<Canvas>
  <Story id="core-tooltip--without-arrow" />
</Canvas>

### Hide icon

Use `hideIcon` prop if you'd like to display the tooltip without state icon.

<Canvas>
  <Story id="core-tooltip--without-icon" />
</Canvas>

### Custom content

Use the `content` prop to pass either a simple string or a React component.

<Canvas>
  <Story id="core-tooltip--custom-content" />
</Canvas>

### Placement

Use the `placement` prop to position the tooltip. The tooltip will flip and shift if there isn't enough space for it to be visible.

<Canvas>
  <Story id="core-tooltip--placement" decorators={[withFlexGap]} />
</Canvas>

### Delay before shown

Use `enterDelay` prop to change the delay before the tooltip is shown. Defaults to 300ms.

  <Story id="core-tooltip--delay" decorators={[withFlexGap]} />
</Canvas>

## Accessibility

We do not recommend that you include interactive elements within a tooltip as this is inaccessible since tooltips do not receive focus. If you need to include focusable content like buttons or links, consider using the overlay component.
