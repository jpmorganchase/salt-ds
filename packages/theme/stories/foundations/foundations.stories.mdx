## Foundations

At the core of our CSS Variables are what we call 'Foundations' and have come from our design team.

Foundations are our collection of design tokens which make up the building blocks of our theme.

### Token Name Convention

Foundation design tokens should generally be in the format `--uitk-<foundation name>-<variant>-<scale>-<density>`

1. Always begin with `--uitk-`
2. Follow this by the foundation name, e.g. `---uitk-color-`, `---uitk-shadow-`, `--uitk-spacing-`
3. Append the variant if applicable, e.g. `--uitk-color-blue-`, `--uitk-shadow-elevation-`, `--uitk-spacing-toolbar-`
4. The suffix of the token is usually the scale or rank, e.g. `--uitk-color-blue-700`, `--uitk-shadow-elevation-3`
5. For tokens that are scoped by density\* the suffix should be the suitable density for the defined value, e.g. `--uitk-spacing-toolbar-touch`

\* Here, it is best practice to avoid directly using the density invariant values, and instead use the -unit token. Replace the `-<density>` suffix with `-unit`, e.g. `--uitk-spacing-toolbar-unit` to get the appropriate value in actual usage.

## Using Foundations

Wrap the foundation you wish to use inside a `var()` function. Most commonly, foundations are used when referenced by characteristics, for example:

```css
.uitk-light,
.uitk-dark {
  --uitk-actionable-font-weight-primary: var(--uitk-typography-weight-bold);
}
```

where the **actionable characteristic** `--uitk-actionable-font-weight-primary` references the variable `--uitk-typography-weight-bold` from the **typography foundation**.

However, foundations can also be used directly if there is no appropriate characteristic provided for your use case. For example, you could color your headings green:

```css
.h1 {
  color: var(--uitk-color-green-100);
}
```

or, even inline!

```html
<h1 style={{ color: "var(--uitk-color-green-100)" }}>Example</h1>
```

For size and spacing foundations, it is highly common to use these within a calculation for width, height, paddings, margings, etc. For example:

```css
.uitkLogo {
  height: calc(1.5 * var(--uitk-spacing-unit)));
}
```
