# Overlayable

Components that may form a hierarchy and stack upon or beneath other elements of the UI.

## CSS Variables

### Shadow (Elevation)

| Shadow (Elevation) | Description                                                                      |
| ------------------ | -------------------------------------------------------------------------------- |
| Scroll             | Dropshadow found at the top of an overflowing scrollable area (e.g. Comments)    |
| Border region      | Dropshadow found around application regions (e.g. App Header, Drawer)            |
| Default            | Dropshadow default (e.g. Card)                                                   |
| Hover              | Dropshadow found around a component using the default (e.g. Card) on hover state |
| Popout             | Dropshadow found around temporarily overlaying elements (e.g. Tooltip)           |
| Drag               | Dropshadow found around draggable objects on drag (e.g. Tabs)                    |
| Modal              | Dropshadow found around modal windows (e.g. Dialog)                              |
