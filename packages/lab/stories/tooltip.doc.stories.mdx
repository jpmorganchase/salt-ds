# Tooltip

A Tooltip provides guidance to the user when something requires more explanation or context, or to provide more information about an error or warning condition. A Tooltip will typically be displayed after the userâ€™s mouse hovers over the target element for a certain amount of time.

## Default

<Canvas>
  <Story id="core-tooltip--default" />
</Canvas>

## API

```
import { Tooltip } from "@salt-ds/core";
```

## Features

### Status

Use the `status` prop to choose between `error`, `warning`, `success` and `info`. Defaults to `info`.

<Canvas>
  <Story name="Tooltip Status" decorators={[withFlexGap]}>
    <Tooltip content="I am a tooltip" status="info">
      <Button>Info</Button>
    </Tooltip>
    <Tooltip content="We found an issue" status="error">
      <Button>Error</Button>
    </Tooltip>
    <Tooltip content="Are you sure" status="warning">
      <Button>Warning</Button>
    </Tooltip>
    <Tooltip content="Well done" status="success">
      <Button>Success</Button>
    </Tooltip>
  </Story>
</Canvas>

### Hide arrow

Use `hideArrow` prop if you'd like to display the tooltip without arrow tip. Defaults to false

<Canvas>
  <Story name="Tooltip hideArrow" decorators={[withFlexGap]}>
    <Tooltip content="I am a tooltip" hideArrow>
      <Button>Without Arrow</Button>
    </Tooltip>
  </Story>
</Canvas>

### Hide icon

Use `hideIcon` prop if you'd like to display the tooltip without state icon. Defaults to false

<Canvas>
  <Story name="Tooltip hideIcon" decorators={[withFlexGap]}>
    <Tooltip content="I am a tooltip" hideIcon>
      <Button>Without StatusIcon</Button>
    </Tooltip>
  </Story>
</Canvas>

### Custom content

Use the `content` prop to pass either a simple string or a react component.

<Canvas>
  <Story name="Tooltip Custom" decorators={[withFlexGap]}>
    <Tooltip
      content={
        <>
          <Text styleAs="h3">Tooltip</Text>
          <Text>Props:</Text>
          <ul>
            <li>
              <strong>hideIcon:</strong> true
            </li>
            <li>
              <strong>open:</strong> true
            </li>
            <li>
              <strong>status:</strong> info
            </li>
          </ul>
        </>
      }
      hideIcon
    >
      <Button>Complex content</Button>
    </Tooltip>
  </Story>
</Canvas>

### Placement

Use the `placement` prop to position the tooltip. Defaults to `right`. The tooltip will flip and shift if there isn't enough space for it to be visible.

<Canvas>
  <Story name="Tooltip Placement" decorators={[withFlexGap]}>
    <Tooltip content="I am a tooltip" placement={"top"}>
      <Button>Top</Button>
    </Tooltip>
    <Tooltip content="I am a tooltip" placement={"bottom"}>
      <Button>Bottom</Button>
    </Tooltip>
    <Tooltip content="I am a tooltip" placement={"left"}>
      <Button>Left</Button>
    </Tooltip>
    <Tooltip content="I am a tooltip" placement={"right"}>
      <Button>Right</Button>
    </Tooltip>
  </Story>
</Canvas>

### Delay before shown

Use `enterDelay` prop to change the delay before the tooltip is shown. Defaults to 300ms

<Canvas>
  <Story name="Tooltip enterDelay" decorators={[withFlexGap]}>
    <Tooltip content="I am a tooltip" enterDelay={100}>
      <Button>100ms</Button>
    </Tooltip>
    <Tooltip content="I am a tooltip">
      <Button>300ms</Button>
    </Tooltip>
    <Tooltip content="I am a tooltip" enterDelay={500}>
      <Button>500ms</Button>
    </Tooltip>
  </Story>
</Canvas>
