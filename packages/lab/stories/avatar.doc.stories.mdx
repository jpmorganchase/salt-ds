# Avatar

An Avatar is a small circular graphical representation of a person through a personal profile picture or name initials.

## Default Avatar

<Canvas>
  <Story name="Default Avatar" decorators={[withFlexGap]}>
    <Avatar size="small" />
    <Avatar size="medium" />
    <Avatar size="large" />
  </Story>
</Canvas>

## Image Avatar

Reference your image in the `src` prop

<Canvas>
  <Story name="Image Avatar" decorators={[withFlexGap]}>
    <Avatar alt="Logo" size="small" src="/docs/assets/placeholder.svg" />
    <Avatar alt="Logo" size="medium" src="/docs/assets/placeholder.svg" />
    <Avatar alt="Logo" size="large" src="/docs/assets/placeholder.svg" />
  </Story>
</Canvas>

## Initials Avatar

You can pass the initials as children

<Canvas>
  <Story name="Initials Avatar" decorators={[withFlexGap]}>
    <Avatar size="small">SB</Avatar>
    <Avatar size="medium">SB</Avatar>
    <Avatar size="large">SB</Avatar>
  </Story>
</Canvas>

## --uitkAvatar CSS Custom Property API

The default foreground and background colors of the Avatar are provided by the theme variables. See Theme documentation for detailed usage guidance.

The CSS custom properties below are consumed by Avatar, but not defined by Avatar. They can be defined via a container or
a custom class name to override avatar styling. They will always take precedence over default styles, whether from theme variable
or declared locally. Again, see Theme documentation for guidance on when to use variables from the Custom Property API.
