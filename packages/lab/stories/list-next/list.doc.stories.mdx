## How to use

### When to use List

- To present a set of selectable options.
- To define attributes for an onward action such as filtering a data grid, grouping reports or selecting templates.

### When not to use List

- To provide navigation within an application. Use vertical navigation instead.
- To navigate options with hierarchical parent-child relationship. Instead, use components that suit your needs such as: tree, cascading menu, or menu button.
- To execute commands, or trigger the display of modal windows or dialog boxes.
- To select among a small number of options. Consider checkbox or radio button instead.

## Accessibility

List Next is accessible by assistive technologies like screen readers, and
provides full WAI-ARIA support.

### Keyboard Interactions

| Key            | Function                                                                                                                                             |
| -------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| Tab            | If focus is above the List, Tab key moves focus into:<br/>The last selected item;<br/>The last item with focus or;<br/>The first item in the List    |
| Shift + Tab    | If focus is below the List, Tab key moves focus into the list item which last had focus; or the first item in the List                               |
| Enter or Space | When focus is on a listed item, Enter or Space keys press, select the item and deselects the previously selected item.                               |
| Arrow up       | Moves focus to the previous item in the List <br /> If focus is on the first item, the next UpArrow key press should NOT wrap focus to the last item |
| Arrow down     | Moves focus to next item in the List. <br /> If focus is on the last item, the next DownArrow key press should NOT wrap focus back to the first item |
| Home           | Moves focus to the first list item.                                                                                                                  |
