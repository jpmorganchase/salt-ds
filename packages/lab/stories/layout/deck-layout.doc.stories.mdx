# Deck layout

The deck layout defines pages of content that appear within the same specified region, one at a time.

It allows motion customisation (`slide` or `fade`) and a custom direction for the animations (`horizontal` or `vertical`).

This layout component also allows you to assign the initial page to be rendered via the `activeIndex` prop.

<Canvas>
  <Story id="lab-layout-decklayout--default-deck-layout" />
</Canvas>

## How you can use deck layout

Use this component by passing it a collection of children (pages). You may also choose to control the movement between pages by using buttons or tabs.

## API

```
import { DeckLayout } from "@jpmorganchase/uitk-lab";
```
