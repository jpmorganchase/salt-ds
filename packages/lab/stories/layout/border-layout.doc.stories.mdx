# Border Layout

Defines the main content regions of an application, region or widget, such as a footer, header or side navigation.
The border layout is built using [Grid layout](?path=/docs/layout-docs-core-gridlayout--page) and it is intended for laying out pages where there is at least a main area with a header and footer.
