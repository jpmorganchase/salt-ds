# Toolbar

Toolbar is a flexible container for controls.

<br />

Simple `Toolbar` with 4 `Buttons`. Note that Button text is not displayed if Button
contains an Icon.

```jsx
<Toolbar>
  <Button variant="secondary">
    <ExportIcon /> Export
  </Button>
  <Button variant="secondary">
    <ShareIcon /> Share
  </Button>
  <Button variant="secondary">
    <NotificationIcon /> Set Alerts
  </Button>
  <Button variant="secondary">
    <TearOutIcon /> Expand
  </Button>
  <Button variant="secondary">More</Button>
</Toolbar>
```
