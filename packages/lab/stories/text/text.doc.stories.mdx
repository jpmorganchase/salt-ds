# Text

We've created a base Text component (that contains our key typography styles and functionality) and more typography components which derive from it: H1, H2, H3, H4, Div, Span, P, LabelCaption, Code etc. Using the Text components will guarantee that the typography styles are consistent throughout the app.

# API

```
import { Text, H1, H2, H3, H4, Div, Span, P, LabelCaption, HelpText, Code, Figure1, Figure2, Figure3 } from "@jpmorganchase/uitk-lab";
```

## Text components

We've created a set of components that are most commonly used, but it's easy to generate your own by passing the tag you want to be returned into the `elementType` prop. The base Text component is generating a `div` by default
