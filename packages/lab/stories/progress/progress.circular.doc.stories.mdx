# Circular Progress

A circular progress indicator gives the user an understanding of how long a system operation will take, indicated by how complete the circle is. It should be used when the operation will take more than a second to complete.

Improve accessibility by customizing the ariaLabel prop to provide additional context about what is loading. For example:

```
ariaLabel = “loading settings panel 0%”
```
