# Progress

A progress indicator gives the user an understanding of how long a system operation will take. It should be used when the operation will take more than a second to complete. Two variants are available to accommodate different layout constraints: Linear and Circular.

## Accessibility

Improve accessibility by customizing the ariaLabel prop to provide additional context about what is loading. For example:

```
ariaLabel = “loading settings panel 0%”
```
