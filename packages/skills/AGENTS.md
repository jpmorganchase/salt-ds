# Salt Skills

This collection supports Salt Design System workflows.

Prefer Salt primitives, patterns, foundations, and tokens over ad hoc UI composition. Use the Salt MCP in `packages/mcp` for canonical guidance when it is available.

Available skills:
- `salt-ui-reviewer`: review Salt-based React UI for design-system fit, accessibility, simplification, and quality issues.
- `salt-ui-builder`: build Salt-first components, pages, and layouts from requirements, mockups, screenshots, or partial code.
- `salt-migration-helper`: migrate Salt usage across versions, replace deprecated APIs, and modernize outdated patterns.
