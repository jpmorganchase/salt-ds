# MCP Docs

Use this folder by audience:

- [`consumer/`](./consumer)
  - current external-consumer setup, workflow, troubleshooting, install, and project-conventions docs
- [`maintainers/`](./maintainers)
  - current maintainer guidance, strategy, and execution tracking
  - [`maintainers/ai-v1-implementation-backlog.md`](./maintainers/ai-v1-implementation-backlog.md) is the sole active execution backlog
  - [`maintainers/consumer-ai-roadmap.md`](./maintainers/consumer-ai-roadmap.md) is the short strategic roadmap
  - [`maintainers/maintaining-salt-ai-tooling.md`](./maintainers/maintaining-salt-ai-tooling.md) is the architectural maintainer guide
- [`archive/`](./archive)
  - completed planning and historical implementation context

Active current-state docs should stay in `consumer/` or `maintainers/`.
Finished plans should move to `archive/` instead of sitting beside active guidance.
