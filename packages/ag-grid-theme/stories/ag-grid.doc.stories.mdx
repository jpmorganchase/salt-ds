# AG Grid theme

AG Grid theme gives you the ability to add our theme styles to AG Grid. The package contains CSS files only, and is applicable for AG Grid versions 25, 26 and 27.

The AG Grid theme package provides two different themes for your use case:

- _UITK AG Grid theme_ contains CSS files with a theme that resembles the Toolkit V1 Data Grid component; use this when you would like to keep the Toolkit V1 theme.
- _Salt AG Grid theme_ contains CSS files with a theme in line with Salt Design System; use this when you would like to go with our latest design (the Salt theme).

Consider using the [Salt Design System Grid component](https://saltdesignsystem-storybook.pages.dev/?path=/docs/documentation-data-grid-data-grid--page) built in-house, in alignment with the Salt theme, if you would prefer a component that is not related to AG Grid.

For any queries that are not related to CSS files, please refer to [AG Grid Documentation](https://www.ag-grid.com/javascript-data-grid/grid-api/).

## Configuring AG Grid theme

To install the `@salt-ds/ag-grid-theme` package, run the command appropriate to your environment:

```
yarn add @salt-ds/ag-grid-theme
```

```
npm install @salt-ds/ag-grid-theme
```

1. Follow these instructions to [use _UITK AG Grid theme_](#uitk)
2. Follow these instructions to [use _Salt AG Grid theme_](#salt)

### <a name="uitk"></a>Installing UITK AG Grid theme

To import the UITK AG Grid theme, run the following command:

```
import '@salt-ds/ag-grid-theme/uitk-ag-theme.css';
```

#### CSS class names for UITK AG Grid theme

For light mode,

```
.ag-theme-uitk-high-light
.ag-theme-uitk-medium-light
.ag-theme-uitk-low-light
.ag-theme-uitk-touch-light
```

For dark mode,

```
.ag-theme-uitk-high-dark
.ag-theme-uitk-medium-dark
.ag-theme-uitk-low-dark
.ag-theme-uitk-touch-dark
```

### <a name="salt"></a>Installing Salt AG Grid theme

To import the Salt AG Grid theme, run the following command:

```
import '@salt-ds/ag-grid-theme/salt-ag-theme.css';
```

#### CSS class names for Salt AG Grid theme

For light mode, add one the following class names to the element wrapping ag-grid:

```
.ag-theme-salt-high-compact-light
.ag-theme-salt-high-light
.ag-theme-salt-medium-light
.ag-theme-salt-low-light
.ag-theme-salt-touch-light
```

For dark mode, add one the following class names to the element wrapping ag-grid:

```
.ag-theme-salt-high-compact-dark
.ag-theme-salt-high-dark
.ag-theme-salt-medium-dark
.ag-theme-salt-low-dark
.ag-theme-salt-touch-dark
```

For variants, add one the following class names to the element wrapping ag-grid:

```
.ag-theme-salt-variant-secondary
.ag-theme-salt-variant-zebra
```

For editable fields, add this class name to the `cellClass` array in your column definition:

```
.editable-cell
```

For fields displaying numeric values add this class name to the `cellClass` array in your column definition:

```
.editable-cell
```

## Examples

Here is an example using _UITK AG Grid theme_:

Wrap `AgGridReact` component with a `div` and the `className` prop you prefer,

```
<div className="ag-theme-uitk-touch-light" style={{ height: 800, width: 600 }}>
  <AgGridReact {...props} />
</div>

```

Use the 'Salt' or 'UITK' button in each example below to toggle between _Salt AG Grid theme_ and _UITK AG Grid theme_.

The class names are:

- `ag-theme-salt-variant-secondary`
- `ag-theme-salt-variant-zebra`
