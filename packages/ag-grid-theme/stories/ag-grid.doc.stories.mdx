# AG Grid Theme

AG Grid Theme provides CSS theme files for AG Grid. The package contains CSS files only, and should be applicable for all AG Grid versions.

AG Grid Theme package provides 2 different themes for your use case:

1. _UITK AG Grid theme (Old theme)_: contains CSS files with theme that resembles Toolkit V1 Data Grid component, use this when you would like to keep the Toolkit V1 theme.
2. _Salt AG Grid theme (New theme)_: contains CSS files with theme in line with Salt Design System, use this when you would like to go with our latest design (the SDS theme).

Consider using [Salt Design System Grid component](https://uitk.pages.dev/?path=/docs/documentation-grid-grid--page) built in-house, in alignment with SDS theme, if you would prefer non AG Grid related component.

For any queries non CSS files related, please refer to [AG Grid Documentation](https://www.ag-grid.com/javascript-data-grid/grid-api/).

## Configuring AG Grid Theme

Run command below to install `@jpmorganchase/uitk-ag-grid-theme` package.

```
yarn add @jpmorganchase/uitk-ag-grid-theme;
```

```
npm install @jpmorganchase/uitk-ag-grid-theme;
```

1. Follow these instructions to [use _UITK AG Grid theme (Old theme)_](#uitk)
2. Follow these instructions to [use _Salt AG Grid theme (New theme)_](#salt)

### <a name="uitk"></a>Installing UITK AG Grid Theme (Old theme)

Use the following call to import AG Grid Theme.

```
import '@jpmorganchase/uitk-ag-grid-theme/uitk-ag-theme.css';
```

#### CSS classnames for UITK AG Grid theme

For light mode,

```
.ag-theme-uitk-high-light
.ag-theme-uitk-medium-light
.ag-theme-uitk-low-light
.ag-theme-uitk-touch-light
```

For dark mode,

```
.ag-theme-uitk-high-dark
.ag-theme-uitk-medium-dark
.ag-theme-uitk-low-dark
.ag-theme-uitk-touch-dark
```

### <a name="salt"></a>Installing Salt AG Grid Theme (New theme)

Use the following call to import AG Grid Theme.

```
import '@jpmorganchase/uitk-ag-grid-theme/odyssey-ag-theme.css';
```

#### CSS classnames for Salt AG Grid theme

For light mode,

```
.ag-theme-odyssey-high-light
.ag-theme-odyssey-medium-light
.ag-theme-odyssey-low-light
.ag-theme-odyssey-touch-light
```

For dark mode,

```
.ag-theme-odyssey-high-dark
.ag-theme-odyssey-medium-dark
.ag-theme-odyssey-low-dark
.ag-theme-odyssey-touch-dark
```

## Examples

Here is an example using _UITK AG Grid theme (Old theme)_:

Wrap `AgGridReact` component with a `div` and the `className` prop you prefer,

```
<div className="ag-theme-uitk-touch-light" style={{ height: 800, width: 600 }}>
  <AgGridReact {...props} />
</div>

```

Use "New theme" button in each example below to toggle between _UITK AG Grid theme (Old theme)_ and _Salt AG Grid theme (New theme)_.
